# psql

Connecting

`psql -h localhost -p 5432 -U postgres -d database`

**Always use semicolons at the end of commands.**

Help

`\?` and `\h`

Exiting psql

`\q`


## in psql

`create database dbname`

Connect to a database

`\c dbname`

Current directory

`\! pwd`

Change directory

`\cd dirname`
