# Express.js

## Starting a new project

`npm init -y` creates `package.json` with default values

*main* file will be `index.js`. `touch` another file such as `server.js` and it will be the *main* file.

## .gitignore

```
*~
\#*
.\#*

node_modules/
```


## Required packages

Run `npm install`

* `express`
* `body-parser`
* `sqlite3`
* `md5`
